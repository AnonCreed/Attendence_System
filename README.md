# Face Attendance System

A Python-based face recognition attendance system using OpenCV and face_recognition.

## Features
- Face enrollment via webcam or image file
- Face recognition for attendance
- Encoding-based identity matching
- Local storage (no cloud)

## Requirements
- Python 3.10 / 3.11 recommended
- OpenCV
- face_recognition
- dlib

## Run
```bash
python enrollment/enroll.py
python attendence/recognize.py
